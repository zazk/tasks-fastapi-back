
*Tasks API FastApi*
---
Python Version
```
Python 3.11.6
```
Install Dependences 
```
pip install -r requirements.txt
```
Launch Dev
```
fastapi dev main.py
```
Run in
```
http://127.0.0.1:8000/
```
Swagger
```
http://127.0.0.1:8000/docs
```

**Documentation**: <a href="https://fastapi.tiangolo.com" target="_blank">https://fastapi.tiangolo.com</a>
